Script Upload Guide

Uploading Your Scripts

1 Prepare Your Script

Ensure your script meets our quality standards:

-- Script Name: MyAwesomeScript
-- Author: YourUsername
-- Version: 1.0.0
-- Description: Brief description of what your script does

local MyScript = {
    Name = "MyAwesomeScript",
    Author = "YourUsername",
    Version = "1.0.0"
}

-- Your code here...

2 Add Required Metadata

Include these essential fields:

  • Script name
  • Version number
  • Description
  • Supported games
  • Required permissions

3 Submit for Review

Upload your script through our submission system:

  1. Go to the Scripts page
  2. Click "Upload New Script"
  3. Fill in the submission form
  4. Upload your script file
  5. Submit for review

Metadata & Tags

Proper metadata helps users find your script:

{
    "name": "MyScript",
    "version": "1.0.0",
    "author": "YourName",
    "description": "Script description",
    "tags": ["farming", "automation", "gui"],
    "games": ["GameID1", "GameID2"],
    "permissions": ["http", "filesystem"]
}

Version Control

Managing script versions:

  • Use semantic versioning (MAJOR.MINOR.PATCH)
  • Document changes in update logs
  • Maintain backwards compatibility
  • Test updates thoroughly
